# 🧮 Simple Calculator App

A simple calculator built with **C# (WinForms)** that supports basic arithmetic operations.

---

## ✨ Features
- ➕ Addition  
- ➖ Subtraction  
- ✖ Multiplication  
- ➗ Division (with zero check)  
- ± Negate values  
- , Decimal support  
- ← Backspace  
- π Constant button  
- AC Clear all
